## Installation

#### Pre-Requisites
- Node JS : 20.x.x

[:arrow_up: Back to top](#index)

#### Installing dependencies

- Angular CLI: 17.3.8
- TypesSript: ~5.4.2
- TailwindCSS: ^3.4.3
- PostCSS: ^8.4.38
- Autoprefixer: ^10.4.19
- crypto-js: ^4.2.0
- moment: ^2.30.1
- rxjs: ~7.8.0
- SweetAlert2: ^11.11.0

#### Download and setup

- Clone repo

- Install node dependencies
  ```
  $ npm install
  ```
- Start local server
  ```
  $ ng serve
  ```
